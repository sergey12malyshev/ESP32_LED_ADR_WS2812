# ESP32_LED_ADR
Тестовый проект для ESP32. Доработанная демоверсия примера DemoReel100 управления адресной LED-лентой

## Инструментарий
*PlatformIO* + *VS Code*

## Board
- ESP32_DevKitc_V4
- WS2812

PinOut: https://www.electronshik.ru/news/show/12935?from=terraelectronica
